    11. Hydraulic Tables Check

    11.1 Loading¶

    Loading the _hydprop_check GIS files into QGIS will load in data from the "_1d_ta_tables_check.csv" file for plotting purposes. The _hydprop_check is a standard GIS files that can be loaded into QGIS in the standard way.

    The data from the "_1d_ta_tables_check.csv" file is only initially partially loaded and only completely loaded when trying to plot from the file for the first time. This is so the initial loading of the GIS file is fast and only loads the tabular data if it is needed for plotting. This also means that the first time any plotting is done, it may take a little longer than subsequent plots, as the data from the "_1d_ta_tables_check.csv" file is being loaded for the first time.

    Only the channel properties are available for plotting from the "_1d_ta_tables_check.csv" file as the _hydprop_check GIS layer only contains channel features. The cross-section tables are not available.

    The tabular data from the "_1d_ta_tables_check.csv" file is linked to the GIS layer and will be closed when the GIS layers are closed.

    11.2 Plotting¶

    The hydraulic properties check format is supported in the following plot types:

    • Section Plot

    The hydraulic properties check format only supports plotting using the selection tool. See the Plot Window documentation for more information on plotting.
