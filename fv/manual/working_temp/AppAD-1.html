F.4.1 Science

The following sub-sections describe the assignment of a tracer model options: Constant, decay and settling.

F.4.1.1 Conservative

A tracer is simply advected and dispersed.

F.4.1.2 Decay

This model allows for the linear temporal decay of a tracer.

\[\begin{equation} \tag{F.1} \end{equation}\]

                DO jtra = 1,Ntrace
                    j = jtra1 + jtra - 1
                    C = MAX(0.,Ukz(j,2,i3)) / h
                    S(j,i3) = S(j,i3) - Kd(jtra) * C * dZ(i3)
                END DO

F.4.1.3 Settling

This model allows for the settling of a tracer.

\[\begin{equation} \tag{F.2} \end{equation}\]

                DO jtra = 1,Ntrace
                    j = jtra1 + jtra - 1
                    C = MAX(0.,Ukz(j,2,i3)) / h
                    Sbed = ws(jtra) * C
                    Sbed = MIN(Sbed, theta_adv_flux*C*dZ(i3)/dt_)
                    S(j,i3) = S(j,i3) - Sbed
                END DO

F.5 Horizontal Mixing

F.5.1 Science

The following sub-sections describe the scalar mixing model options: none, constant, Elder, Smagorinsky and warmup. These models set or compute the scalar diffusivity \(K^{AD}_d\) (also referred to as eddy diffusivity in the literature) that is multiplicative to the diffusive terms in the scalar conservation equations.

\[\begin{equation} \nabla \cdot \left( K^{AD}_d \nabla \phi \right) \tag{F.3} \end{equation}\]

Where:
- \(K^{AD}_d\) = scalar diffusivity \([\text{m}^2\!/\text{s}]\)
- \(\phi\) = scalar concentration \([\text{mass}\!/\text{m^3}]\)

F.5.1.1 None

This model sets the value of \(K^{AD}_d\) to be zero.

F.5.1.2 Constant

This model sets the value of \(K^{AD}_d\) to be constant.

F.5.1.3 Elder

        ! CALCULATE DEPTH AVERAGED VELOCITY
        IF (hL>hWet.AND.hR>hWet) THEN
            DO n = 0,NL(i2)-1
                u = u + 0.5*(UL(2,i3+n)/hL*dZ(1,i3+n)+UR(2,i3+n)/hR*dZ(2,i3+n))
                v = v + 0.5*(UL(3,i3+n)/hL*dZ(1,i3+n)+UR(3,i3+n)/hR*dZ(2,i3+n))
            END DO
            u = u / h
            v = v / h
        END IF
        umag = SQRT(u**2+v**2)+small
        Dl = MAX(Ds_coef(1,i2),small)
        Dt = MAX(Ds_coef(2,i2),small)
        IF (bot_drag_mod==1) THEN ! MANNING's ROUGHNESS
            f = MAX(h**0.83333333 * SQRT(g) * bot_drag_coeff(i2) / umag, 0.)
        ELSEIF (bot_drag_mod==2) THEN ! ROUGHNESS LENGTH / LOG-LAW
            f = MAX(kappa * h / MAX(log(11. * h / bot_drag_coeff(i2)) * umag, small), 0.)

        Dl = MAX(Dl, Ds_limits(1,i2) / Dtmp)
        Dl = MIN(Dl, Ds_limits(2,i2) / Dtmp)
        Dt = MAX(Dt, Ds_limits(1,i2) / Dtmp)
        Dt = MIN(Dt, Ds_limits(2,i2) / Dtmp)
        Ds(1,i3) = (Dl*u**2 + Dt*v**2)*f
        Ds(2,i3) = (Dl-Dt)*u*v*f
        Ds(3,i3) = (Dl*v**2 + Dt*u**2)*f
        DO n = 1,NL(i2)-1
            Ds(:,i3+n) = Ds(:,i3)
        END DO

F.6 Computational Timestep

F.6.1 Context

F.6.2 Science

F.6.2.1 Model Timestep

Timestep calculations for advection dispersion (in either 2D and 3D) include an addition scalar diffusivity criterion \(\Delta t_s\).

\[\begin{equation} \Delta t_s \;\leq\; \frac{(L^*)^2}{|D_{xx} n_x| + |D_{yy} n_y|} \tag{F.9} \end{equation}\]

Where:
- \(D_{xx}\) = diffusivity in the \(x\)-direction \([\text{m}^2\!/\text{s}]\)
- \(D_{yy}\) = diffusivity in the \(y\)-direction \([\text{m}^2\!/\text{s}]\)
- \(n_x, n_y\) = components of the face unit normal

AD With 2D Hydrodynamics

The maximum allowable internal and external timesteps for advection dispersion in 2D are presented in Equations (F.10) and (F.11) respectively. These are consistent with the 2D HD only equations presented in Appendix D.6.1.2 with the addition scalar diffusivity criterion \(\Delta t_s\).

\[\begin{equation} \Delta t_{\max\_internal} \;=\; \min\!\left( \frac{1}{\tfrac{1}{\Delta t_u} + \tfrac{1}{\Delta t_d}},\; \frac{1}{\tfrac{1}{\Delta t_u} + \tfrac{1}{\Delta t_s}} \right) \tag{F.10} \end{equation}\]

\[\begin{equation} \Delta t_{\max\_external} \;=\; \min\!\left( \frac{1}{\tfrac{1}{\Delta t_c} + \tfrac{1}{\Delta t_d}},\; \frac{1}{\tfrac{1}{\Delta t_u} + \tfrac{1}{\Delta t_s}} \right) \tag{F.11} \end{equation}\]

The resulting \(\Delta t_{\max\_internal}\) and \(\Delta t_{\max\_external}\) are applied in Equations (D.9) and (D.10) when using the global CFL command. If the split commands CFL Internal and CFL External are used, then they are applied in Equations (D.11) and (D.12), as described in Appendix D.6.1.2.

AD With 2D Hydrodynamics

The maximum allowable internal and external timesteps for advection dispersion in 3D are presented in Equations (F.15) and (F.12) respectively. These are consistent with the 3D HD only Equations (D.7) and (D.8) with the addition of the \(\Delta t_s\) term.

\[\begin{equation} \Delta t_{\max\_external} \;=\; \min\!\left( \frac{1}{\tfrac{1}{\Delta t_c} + \tfrac{1}{\Delta t_d(2D)}},\; \frac{1}{\tfrac{1}{\Delta t_u(2D)} + \tfrac{1}{\Delta t_s(2D)}} \right) \tag{F.12} \end{equation}\]

\[\begin{equation} \Delta t_{\max\_internal-h} \;=\; \min\left( \frac{1}{\tfrac{1}{\Delta t_u(i)} + \tfrac{1}{\Delta t_d(i)}},\;\frac{1}{\tfrac{1}{\Delta t_u(i)} + \tfrac{1}{\Delta t_s(i)}} \right) \tag{F.13} \end{equation}\]

\[\begin{equation} \Delta t_{\max\_internal-v} \;=\; \min\left( \frac{\min\!\big(\Delta z^{+}_k,\;\Delta z^{-}_k\big)}{|w_k|} \right) \tag{F.14} \end{equation}\]

\[\begin{equation} \Delta t_{\max\_internal} \;=\; \min\!\left( \Delta t_{\max\_internal-h},\; \Delta t_{\max\_internal-v} \right) \tag{F.15} \end{equation}\]

Where:

  • \(\Delta t_{\max\_internal-h}\) maximum allowable horizontal internal timestep (s)

  • \(\Delta t_{\max\_internal-v}\) maximum allowable vertical internal timestep (s)

  • \(\Delta t_u(i),\, \Delta t_d(i),\, \Delta t_s(i)\) are maximum allowable timestep limits at each 3D horizontal cell face \(i\) (s)

  • \(\Delta t_u(k)\) is the maximum allowable advective timestep limit at each 3D vertical cell face \(k\) (s)

  • \(\Delta z^{+}_k\) = thickness of the cell above face \(k\) (m)

  • \(\Delta z^{-}_k\) = thickness of the cell below face \(k\) (m)

  • \(w_k\) = vertical velocity at face \(k\) (m/s)

The resulting \(\Delta t_{\max\_internal}\) and \(\Delta t_{\max\_external}\) are applied in Equations (D.9) and (D.10) when using the global CFL command. If the split commands CFL Internal and CFL External are used, then they are applied in Equations (D.11) and (D.12), as described in Appendix D.6.1.2.

    ! Calculate stability parameter
    stab = ((wtemp - atemp) + 0.61*(atemp + 0.0098*zrefa)*(es - eair)) / &
            (w10 ** 2) / ((1+log10(10./zrefa))**2)
    
    stab = stab * abs(stab)/(abs(stab) + 0.01)
        
    ! Correcting cdref for the stability
    IF (stab < -3.3) THEN ! very stable
        stab_coefw = 0.0
        stab_coef = stab_coefw
    ELSEIF(stab >= -3.3 .AND. stab < 0) THEN ! slightly stable
        stab_coefw = 0.1 + 0.03*stab + 0.9*EXP(4.8*stab)
        stab_coef = stab_coefw
    ELSEIF (stab >= 0.) THEN !  unstable
        stab_coefw = 1.0 + 0.47*SQRT(stab)
        stab_coef  = 1.0 + 0.63*SQRT(stab)
    ENDIF
    cd10 = cd10 * stab_coefw
    ch = ch * stab_coef
    ce = ce * stab_coef
END SELECT
