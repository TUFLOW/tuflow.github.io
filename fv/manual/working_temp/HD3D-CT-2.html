6.3.1 Command Status

Information Only - Commands are specified during 2D HD model construction.

6.3.2 Description

No additional commands are required for 3D simulation beyond those described in Section 5.7. Additional stability calculations are executed on horizontal and vertical cell faces using the specified timestep limits, as described in Appendix B.4.5.3.
