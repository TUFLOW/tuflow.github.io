TUFLOW Classic/HPC Changelog

New releases are typically linked to the release of major new features and/or improvements in the computational hydraulic engines. Whilst some changes in results can be expected between releases due to computational engine improvements, the intent is not to have any changes in results between minor releases. The default settings are also intended to remain unchanged between minor releases, but can vary between major releases to encourage users to use the latest advancements.

Users can revert to the default settings and prior computational engine approaches by using the Defaults command.

Defaults == PRE 2025 ! An example that sets the defaults back to the 2023 release

Some defaults can be reverted without having to change all defaults back to a specific version. For these, see the TUFLOW User Manual
