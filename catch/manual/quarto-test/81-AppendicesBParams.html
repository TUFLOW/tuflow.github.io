Appendix B — Parameters

Following are parameters that can be specified within TUFLOW CATCH via user commands.

Tips for use:

  • Use the search box to dynamically filter the table
  • Sort the filtered (or unfiltered) table by clicking the arrows at the top of each column
  • UD = user defined
  • N/A = not applicable
  • ‘Command’ column links to a description of the corresponding control file command that includes the parameter
  • ‘Science’ column links to the section containing the relevant descriptions
